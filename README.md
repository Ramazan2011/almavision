## 🇷 Русский

### Панель управления умным городом - Алматы

Мониторинг и аналитика города Алматы в реальном времени с ИИ-аналитикой.

#### Запуск проекта

```bash
npm install
npm run dev
```

#### Функции

- **ИИ-аналитика** — Генерация инсайтов через OpenRouter API с разделением на краткосрочные и долгосрочные прогнозы
- **Мультиязычность** — Поддержка русского, казахского и английского языков
- **Тёмная тема** — Переключение между светлой и тёмной темой
- **Автообновление** — Настраиваемый интервал обновления данных (1, 5, 15 минут)
- **ИИ-чат** — Интерактивный чат с ИИ для вопросов о данных города
- **Журнал инцидентов** — Регистрация и отслеживание городских инцидентов
- **Фильтры данных** — Реальное время, исторические данные, прогнозы
- **Отладочный режим** — Введите `debug.enable()` в консоли для активации

#### Технологии

- React + TypeScript
- Vite
- Tailwind CSS
- Recharts
- Leaflet (карты)
- Open-Meteo API (погода)
- OpenRouter API (ИИ)

## 🇰🇿 Қазақша

### Ақылды қала басқару панелі - Алматы

Алматы қаласын нақты уақытта мониторинг және ЖИ-талдау.

#### Жобаны іске қосу

```bash
npm install
npm run dev
```

#### Мүмкіндіктер

- **ЖИ-талдау** — OpenRouter API арқылы инсайттар генерациялау
- **Көптілділік** — Орыс, қазақ және ағылшын тілдерін қолдау
- **Қараңғы тақырып** — Жарық және қараңғы режимдер арасында ауысу
- **Автожаңарту** — Деректерді жаңарту аралығын баптау (1, 5, 15 минут)
- **ЖИ-чат** — Қала деректері туралы сұрақтар қою
- **Оқиғалар журналы** — Қалалық оқиғаларды тіркеу
- **Деректер сүзгілері** — Нақты уақыт, тарихи деректер, болжамдар
- **Түзету режимі** — Консольде `debug.enable()` теріңіз

#### Технологиялар

- React + TypeScript
- Vite
- Tailwind CSS
- Recharts
- Leaflet (карталар)
- Open-Meteo API (ауа райы)
- OpenRouter API (ЖИ)

## 🇬 English

### Smart City Management Dashboard - Almaty

Real-time monitoring and AI-powered analytics for Almaty city.

#### Running the Project

```bash
npm install
npm run dev
```

#### Features

- **AI Insights** — Generate insights via OpenRouter API with short-term and long-term analysis
- **Multilingual** — Support for Russian, Kazakh, and English languages
- **Dark Mode** — Toggle between light and dark themes
- **Auto Refresh** — Configurable data refresh intervals (1, 5, 15 minutes)
- **AI Chat** — Interactive chat with AI for city data questions
- **Incident Log** — Register and track city incidents
- **Data Filters** — Real-time, historical, and predicted data views
- **Debug Mode** — Type `debug.enable()` in console to activate

#### Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Recharts
- Leaflet (maps)
- Open-Meteo API (weather)
- OpenRouter API (AI)
